Privacy Policy

Last updated: June 30, 2026

Proxy Switcher is a Chrome extension that lets users configure Chrome to use a user-provided HTTP, HTTPS, or SOCKS5 proxy.

Data collection

The developer does not collect, transmit, sell, or share user data. The extension does not send proxy settings, proxy credentials, browsing history, visited URLs, traffic content, IP address, analytics, or telemetry to the developer or to third parties.

Locally stored data

The extension may store the following data locally on the user's device for the proxy switching feature:

  • Proxy type
  • Proxy host
  • Proxy port
  • Proxy username
  • Proxy password, stored locally in the Chrome profile
  • User preferences, such as disclaimer confirmation

The proxy password is stored locally in the Chrome profile on the user's device so the proxy can reconnect automatically after a browser restart. It never leaves the device. “Forget saved data” removes it at any time.

Connection test

The extension verifies connectivity by requesting https://www.gstatic.com/generate_204 when the user turns the proxy on and when the user clicks the status line. The request contains no user data, and no checks run while the proxy is off.

Data sharing

The extension does not sell, share, or transfer user data to third parties.

Browsing history and website content

The extension does not collect browsing history, visited URLs, website content, request content, or traffic data. Host permissions are used only to support proxy authentication for the proxy configured by the user.

Remote code

The extension does not use remote code. All JavaScript, HTML, CSS, and image assets are packaged inside the extension.

Delete saved data

Users can delete saved proxy settings and credentials at any time by using the “Forget saved data” button inside the extension.

User responsibility

This extension does not provide proxy servers or network infrastructure. Users are responsible for using their proxy in compliance with applicable laws, website terms, and proxy provider rules.
